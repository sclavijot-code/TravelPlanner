# TravelPlanner
1. Descarga el archivo .zip "SistemaViajes.zip"
2. Descomprime este archivo.
3. En VisualStudioCode abre la carpeta "SistemaViajes"
4. Asegurese de tener descargada la extensión LiveServer, si no la tiene descargada, hagalo a través del apartado de extenciones de VSCode.
5. Haga click derecho sobre el archivo index.html y elija Open with LiveServer
6. Debería observar la ejecución del programa en el navegador
